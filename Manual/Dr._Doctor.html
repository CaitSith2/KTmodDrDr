On the Subject of Dr. Doctor

*coughs* Since when can a bomb give me a prescription?!?

Someone in the bomb factory infected the Bomb with a Deadly Module. Check out the Symptoms, Diagnose the Sickness, Give a Prescription and schedule a Follow-up.

  • Use the Venn Diagram S1CKN3SS, to start off. You use the Table D1A6N0S3 to make a diagnosis, by using the letter or number gained in the Venn Diagram. You check if all the proposed symptoms are in the possible sickness. If they are, this is the sickness the bomb is having. Otherwise, check the next sickness. Rinse and repeat until the correct sickness is found. If the initial sickness starts with a vowel, use the table in normal order, otherwise, in reverse order.
  • Using the Flow Chart M3D1CA710N, figure out which dose is used to treat the Bomb. Every dose is in mg, if not specified otherwise.
  • Find out the Follow-Up date. By using the Table D8TS find the date closest to the date the bomb was started on. But look out for the exception!
